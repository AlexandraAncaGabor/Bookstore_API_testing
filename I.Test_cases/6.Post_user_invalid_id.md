# 6. Post user with invalid email


## Steps:

1. Open Postman.

2. Create a new POST request to: http://localhost:3000/users.

3. Include in body:

{
  "name": "Jane Doe",

  "email": "invalidemail",
  
  "role": "customer"
}

4. Send the request.

## Expected Result:

- Status code: 400
- Response: "Invalid email format"
