3. Post a new review

## Steps:

1. Open Postman.

2. Create a new POST request to: http://localhost:3000/reviews.

3. Include in body:

{
  "book_id": 1,

  "user_id": 2,

  "rating": 4.5,
  
  "comment": "Great book!"
}

4. Send the request.

## Expected Result:

- Status code: 200
- Response: Details of the newly created review including the generated id
