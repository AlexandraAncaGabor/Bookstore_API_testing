# 2. Put a new user

## Steps:

1. Open Postman.

2. Create a new PUT request to: http://localhost:3000/users/{id} (replace {id} with an existing user ID).

3. Include in body:

{
  "name": "Maria Popa",

  "email": "maria.popa@gmail.com",
  
  "role": "customer"
}

4. Send the request.


## Expected Result:

- Status code: 200

- Response: "User updated successfully"
