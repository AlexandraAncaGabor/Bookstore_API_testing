# 7. Post book with invalid rating

## Steps:

1. Open Postman.

2. Create a new POST request to: http://localhost:3000/books.

3. Include in body:

{
  "title": "Test Book",
  "author": "Author Name",
  "genre": "Fiction",
  "published_year": 2026,
  "rating": 7,
  "status": "completed"
}

4. Send the request.

## Expected Result:

- Status code: 400
- Response: "Invalid rating"
